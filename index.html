e-Pantry

A simple solution to keep track of all your groceries.

Tech Used:

Xamarin Android, xml, C#, php, MySQL, Rapberry Pi, Restful API, Python, Weight/temp sensors

Summary:

e-Pantry had a physical component which used sensors to transfer data to Raspberry pi. It used python scripts to update MYSQL database using PHP webservices. For Software component, Xamarin android was used to create an android App with xml and C# used for its workings. Items could be added by scanning there UPC with made an API request and received JSON data which was stored in our database. Adding,Deleting, Editing data is done using php webservices.
